
                    // MoonScanner //
                      
        Simple tool to scann broken links (site, image).


1. Support
  - check broken link
  - generate both good link and bad link report

2. How to
  - Install packages:
        
  ```python
    pip install -r requirements.txt
  ```
        
  - Add urls to sites.txt
    
  - Run command
        
  ```python
    python app.py
  ```
      
  - Two reports: good.txt and bad.txt, including detailed info.
